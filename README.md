# CNS-ZHS-HUB

一个用于归档和发布独立 HTML 页面的通用仓库。本仓库不针对任何特定域名或网站，仅作为 HTML 资源的集中存放与展示平台。

## 项目简介

CNS-ZHS-HUB 旨在为多个独立的 HTML 页面提供版本化归档与静态发布能力。所有页面均为自包含的 HTML 文件，无需后端支持，可直接通过浏览器访问。

本仓库适用于以下场景：
- 归档临时或长期的 HTML 页面
- 存放无需依赖外部服务的独立页面
- 作为静态页面的集中管理入口

## 目录结构

```
/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源文件（如 CSS、JS、图片）
├── index.html      # 仓库首页或导航页面（如有）
└── README.md       # 本文件
```

- `pages/`：按需存放多个 HTML 文件，文件名应具有辨识度。
- `assets/`：存放多个页面可能共用的资源文件，避免重复引用。

## 页面归档说明

- 每个 HTML 页面应保持独立完整，可自包含运行。
- 页面之间的链接应使用相对路径，确保在本地或 GitHub Pages 环境下均可正常访问。
- 不建议在页面中嵌入外部不可控资源，以确保长期可访问性。

## 维护说明

- 欢迎通过 Pull Request 提交新的 HTML 页面或改进现有页面。
- 提交前请确保页面内容合规、无恶意代码、不侵犯他人权益。
- 本仓库不承担页面内容的审核责任，但保留删除不当内容的权利。
- 如发现页面存在兼容性或访问问题，欢迎提交 Issue 反馈。
